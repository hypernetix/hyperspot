# ModKit Errors

Core error types for ModKit.

## Overview

The `cf-modkit-errors` crate provides pure data types for error handling, including:

- RFC 9457 Problem Details (`Problem`)
- Validation error types
- Error catalog support (`ErrDef`)

## Features

- `utoipa`: OpenAPI schema support
- `axum`: Axum integrations for returning problems as HTTP responses
